# Portfolio Website Sprint Plan

## Sprint Goal
Launch a personal portfolio website that reflects my positioning as a technical, system-oriented SEO professional.

## Core Stack
- Next.js (App Router)
- Vercel
- Sanity CMS

## Pages in Scope
- Home
- Experience
- Projects
- Blog
- Blog Detail

## Sprint Duration
7 Days

## Success Criteria
- Website live on Vercel
- Content manageable via Sanity
- SEO-ready foundation implemented
- Clear personal positioning

---

## Day 1 — Setup
- Create GitHub repo
- Initialize Next.js project
- Setup folder structure
- Connect to Vercel
- Setup Sanity project

## Day 2 — CMS Schema
- Create schema: Experience, Project, Blog
- Add SEO fields
- Input dummy content

## Day 3 — Structure & Wireframe
- Define content per page
- Create simple wireframe
- Finalize page hierarchy

## Day 4 — Frontend Build
- Build layout (navbar, footer)
- Build all pages
- Connect Sanity data

## Day 5 — SEO Layer
- Metadata per page
- Open Graph
- Sitemap & robots.txt
- Structured data

## Day 6 — Content Fill
- Add projects (min 3)
- Add blog (min 2)
- Add experience

## Day 7 — QA & Launch
- Check mobile & performance
- Fix bugs
- Deploy production
